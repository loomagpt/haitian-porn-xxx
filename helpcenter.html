Haitian Tube API Documentation

Authentication

POST https://api.haitiantube.com/v1/auth
Content-Type: application/json
{
  "api_key": "your_api_key_here"
}

Endpoints

  • GET /v1/channel - Get channel information
  • GET /v1/videos - List channel videos
  • GET /v1/analytics - Retrieve analytics data
  • POST /v1/upload - Upload a new video
  • GET /v1/comments - Get video comments
  • GET /v1/live - Live stream management

Rate Limits

10,000 requests per day for verified applications. Contact developer support for increased limits.
